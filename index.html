Selected Projects & Case Studies

Each project below represents an end-to-end analytical solution, from problem definition and data engineering to modeling, evaluation, and deployment.

Credit Risk Assessment System

Objective: Predict loan default risk and support data-driven lending decisions.

This project involved building and evaluating multiple supervised machine learning models to predict the probability of loan default. The dataset included borrower demographics, financial history, and loan attributes.

  • Engineered features from raw financial and behavioral variables
  • Compared Logistic Regression, Random Forest, Gradient-based models, and Neural Networks
  • Achieved up to 92% predictive accuracy with strong ROC-AUC performance
  • Focused on interpretability for real-world credit decisions

The final model was deployed as an interactive Streamlit application, allowing users to explore predictions, risk scores, and insights in real time.
