Sine-Type II Half-Logistic Weibull Distribution Framework

Domain: Statistical Research • Probability Theory • Mathematical Modeling

Objective: Design and implement a novel probability distribution for modeling complex real-world phenomena with enhanced flexibility for reliability analysis, survival modeling, and financial data applications.

This research-grade implementation presents the Sine-Type II Half-Logistic Weibull (STIIHLW) distribution through a three-stage transformation process with interactive visualization of statistical functions.

Theoretical Foundation:

  • Base Distribution: Weibull distribution for reliability and lifetime data analysis
  • Type II Half-Logistic Transformation: Enhances skewness control and tail behavior while preserving support
  • Sine Transformation: Introduces non-monotonic hazard functions and multi-modal behavior
  • Mathematical Implementation: Complete derivation of PDF, CDF, survival, and hazard functions
  • Parameter Estimation: Maximum Likelihood Estimation and Bayesian methods for distribution fitting

Visualization Framework:

  • Real-time parameter adjustment with immediate visualization updates
  • Comparative analysis with base Weibull distribution
  • Interactive plots for PDF, CDF, survival, and hazard functions
  • Clean mathematical structure with interpretable parameter effects
  • Mobile-responsive deployment via Streamlit Community Cloud
